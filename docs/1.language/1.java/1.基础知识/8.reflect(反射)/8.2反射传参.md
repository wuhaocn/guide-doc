### 1.反射不定参传递错误：

- 1.1.错误信息

```
警告： 最后一个参数使用了不准确的变量类型的 varargs 方法的非 varargs 调用；
[javac] 对于 varargs 调用，应使用 java.lang.Object
[javac] 对于非 varargs 调用，应使用 java.lang.Object[]，这样也可以抑制此警告
```

- 1.2.多参数解决

```
Class<?> clsBoot = Class.forName("com.xxx.xxx.XXServerBoot", true, ClasspathLoader.classloader);
Method method = clsBoot.getDeclaredMethod("start",  new Class[]{String[].class});
method.invoke(null, new Object[]{args});
```

```
public class XXServerBoot {
    public static void start(String[] args) {
        //...
    }
}
```
